## 1. HTML5 기본 용어
### 1) 태그와 요소
`* 요소 (Element)`

  **HTML 페이지를 구성하는 각 부품**
  
  ex) 제목, 본문, 이미지 등..

- 빈 요소 (Empty)
  - 내용이 없는 요소 (가질 수 없거나 필요X)
  - 속성만 소유, 속성을 통해 전달
  - 사람이 읽는 정보가 아님
 
```html
<h1>Hello HTML5</h1>
<p>즐거운 웹 프로그래밍 입문</p>
<audio></audio>

<!-- 내용無 요소 (빈 요소) -->
<img>
<br>
<hr>
```
<br>
`* 태그`
  **요소를 만들 때 사용하는 작성 방법**
  
  <img src = "https://velog.velcdn.com/images/seizethedai/post/d6fd9151-c8ab-4041-a130-449030424756/image.png" width=500px>

### 2) 속성
`* 속성 (Attribute)`
  **요소의 성질, 특징 정의**
  - 태그(요소)에 추가 정보를 부여할 때 사용 (ex. 이미지 파일 경로, 크기 등)
  `<태그 속성="값">내용</태그>`
